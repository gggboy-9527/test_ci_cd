# frontserver-1 (Vue 3 + Vite)

快速开始（PowerShell）:

```powershell
# 进入项目目录
cd d:\workspace\frontserver_1

# 安装依赖
npm install

# 本地开发
npm run dev

# 构建
npm run build

# 预览构建产物
npm run preview
```

项目包含:

- `src/main.js` - 应用入口
- `src/App.vue` - 根组件
- `src/router` - Vue Router 示例
- `src/store` - Pinia 示例

建议:

- 根据团队需要添加 ESLint/Prettier 配置并统一提交钩子
- 将环境变量放在 `.env` 文件并在 CI 中管理密钥
